5 HTML TAGS

Below is a list of HTML tags and a description of what they do.

  1. <article>
    Use this tag to denote independent, self contained content. Useful for orgnazing content for CSS styling.

  2. <nav>
    Use this tag to specify your website's main navigation links. Note that not all links should be <nav>.

  3. <img>
    This tag lets you imbed an image into your webpage. You can also turn the image into a link by nesting it inside an >a> tag.

  4. <input>
    This tag shows a place where users can enter data. Very important for building a form.

  5. <mark>
    Use this tag for text that needs to be highlighted or marked. It's similar to the italics tag (<i>).
